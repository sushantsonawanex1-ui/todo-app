# 📝 Todo App

A clean and responsive Todo App built using HTML, CSS, and JavaScript. It allows users to add, complete, and delete daily tasks with an easy-to-use interface.

---

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 📱 Responsive design
- ⚡ Fast and lightweight

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
